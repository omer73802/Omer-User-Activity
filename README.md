# Social Media Analytics App

## Project Overview
This interactive analytics application explores user behavior across social media platforms.

The goal of the project is to analyze engagement patterns, platform usage trends, and differences between verified and non-verified users.

The application was built using Streamlit, enabling dynamic filtering and real-time data exploration.

** Live App:**  
[https://omer-social-media-analytics.streamlit.app/](https://omer-user-activity-website.streamlit.app/)


---

##  Dataset
Social Media Users Dataset  

The dataset contains simulated user data including:

- Platform
- Country
- Daily Time Spent (min)
- Verified Account
- Primary Usage
- Date Joined
